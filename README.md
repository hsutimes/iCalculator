# iCalculator
A calculator by myself in Java

***

## 中文说明
### 目的
学习表达式求值后，根据自己的理解，运用Java编写了该计算器。计算器可以完成小数，多位数，带符号数和带括号表达式等复杂的计算。

队列中的符号替代：
>sin：-1.01101
cos：-1.01102
tan：-1.01103
sqrt：-1.01104
加号：-1.01001
减号：-1.01002
乘号：-1.01003
除号：-1.01004
乘方：-1.01005
左括号：-1.01006
右括号：-1.01007
